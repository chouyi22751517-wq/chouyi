這是一個為你設計的「台北私藏風景」導覽網站。作為前端工程師與 UI 設計師，我採用了現代感的磨砂玻璃風格（Glassmorphism），並結合流暢的狀態切換來實現問卷與推薦系統。

這個網頁包含以下特色：

沉浸式視覺：使用大面積的台北街景與自然風景作為設計語彙。

互動問卷：透過三個維度（氛圍、時間、步調）快速分析使用者的旅行人格。

動態推薦：根據問卷結果，從資料庫中過濾出最適合的台北景點，並提供精美的卡片展示。

響應式設計：完美適配電腦與行動裝置。

設計思路說明：
視覺風格 (Visual Style)：採用了流行的 Dark Mode 搭配高對比的 Blue-to-Emerald 漸層色，營造出高端且科技感的旅遊氛圍。

使用者體驗 (UX)：

首頁 (Hero Section)：使用具有呼吸感的縮放背景圖，並以簡短有力的文案引導使用者。

問卷系統 (Quiz)：捨棄傳統的表單，改用大尺寸的按鈕卡片與圖示，減少使用者的填寫負擔，讓測驗過程像在玩心理測驗一樣有趣。

推薦邏輯：根據「氛圍」、「時間」、「體力需求」三個標籤進行權重計算，精準推薦對應景點。

UI 組件：

Lucide Icons：用於增加介面的視覺提示。

Tailwind CSS：實現流暢的 hover 縮放、毛玻璃效果 (backdrop-blur) 與響應式佈局。

狀態管理：使用 React useState 控制「首頁 -> 問卷 -> 結果」的流暢切換。
